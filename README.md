### 2. Beijing PM2.5 Analysis
Analysis project using the UCI Beijing PM2.5 dataset (2010–2014) to identify pollution trends, weather-driven patterns, and high-risk time periods through Python-based preprocessing and Tableau dashboards.
- **View Project**: [GitHub Repository](https://github.com/MilindDevX/SectionD_G12_BeijingPM25Analysis)

### 3. Amazon Indian Sales EDA
A Tableau dashboard project that analyzes Amazon sales data in India to provide actionable insights into sales performance, fulfillment methods, geographical distribution, and product category trends.
- **View Project**: [GitHub Repository](https://github.com/yadavom7345/amazon_indian_sales_EDA)

### 4. IBM HR Analytics EDA
An interactive Tableau dashboard designed to analyze HR data and understand employee attrition at IBM. Uses a comprehensive dataset to uncover factors that influence whether employees leave or stay with the company.
- **View Project**: [GitHub Repository](https://github.com/yadavom7345/IBM_HR_Analytics_EDA)
